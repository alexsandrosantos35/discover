## font-family 

- tipo de fonte de um elemento
- lista de fontes e ordem de prioridade
- inclui fallback font

--------------------------------------------

## Font Family

```css 
p {
font-family: "times New Roman", Times, serif;
}
```
- serif
- sans

----------------------------------------------

## Font Weight

* Peso da fonte

```css
p {
    font-weight: bold;
}
```

------------------------------------------------
## font-style

* estlo da fornte

```css
 span  {
    font-style: italic;
  }
```

------------------------------------------------

## font-size

* tamanho da fonte

```css
 p {
    font-size: 18px;
 }
```
