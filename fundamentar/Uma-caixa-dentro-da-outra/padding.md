## Padding

Espaço interno dos elementos

padding-top|padding-right|padding-bottom|padding-left
values: &lt;length> | &lt;percentage> 

```css

             /* shorthand */
   padding: 12px 16px 10px 4px;
   padding: 12px 16px 0; 
   padding: 8px 16px;
   padding: 8px;
```
    padding pode causar diferenças na largura de um elemento
